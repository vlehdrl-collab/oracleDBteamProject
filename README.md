##프로젝트를 하면서 느낀점##

남는 시간에 복습하는 시간을 많이 가졌던것 같습니다. 명령어를 여러개 만들어서 ppt에 많은 분량을 넣을수도 있었지만, 단순히 분량을 채우는건
별로 의미가 없다고 생각되어서 제가 넣고싶은 기능만 넣어보았습니다.
